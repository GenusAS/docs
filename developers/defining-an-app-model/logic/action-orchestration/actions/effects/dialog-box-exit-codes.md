# Dialog Box Exit Codes

## Boolean Exit Codes

Button | Value    |
-------|----------|
OK     | TRUE     |
Cancel | FALSE    |
Yes    | TRUE     |
No     | FALSE    |

## Integer Exit Codes

Button | Value    |
-------|----------|
OK     | 1        |
Yes    | 2        |
No     | 3        |
Cancel | 4        |
